01 — BEFORE YOU START

시작 전 준비물

아래 항목을 먼저 확인하고 9단계 테스트로 넘어가세요.

  • Google 계정 로그인 가능 여부 Colab은 Google 계정으로 동작합니다. 회사 SSO 계정과 개인 계정 중 어느 것을 사용할지 운영 담당자 안내에 따라 통일해 주세요.
  • AWS 임시 자격 증명 수령 Access Key, Secret Key, Session Token(또는 AssumeRole 안내)을 운영팀으로부터 받으셨는지 확인. 받지 못했다면 먼저 요청해 주세요.
  • 사용 리전(Region) 확인 운영팀이 지정한 리전(예: us-east-1)을 메모해 두세요. 리전이 다르면 모델 호출이 실패합니다.
  • 본인 회사 네트워크에서 진행 교육 당일과 동일한 환경에서 테스트해야 합니다. 모바일 핫스팟이나 집 네트워크에서 통과해도 의미가 없습니다.
  • 약 45분의 여유 시간 확보 중간에 끊기면 자격 증명 만료 등으로 처음부터 다시 해야 할 수 있습니다.

03

AWS 자격 증명 설정

get_caller_identity()에서 Account ID 출력

운영팀에서 전달받은 키를 아래 형식으로 입력합니다. 노트북을 공유하거나 저장할 때 이 셀의 출력은 반드시 지워주세요.

import os
os.environ['AWS_ACCESS_KEY_ID']     = '전달받은 키 입력'
os.environ['AWS_SECRET_ACCESS_KEY'] = '전달받은 시크릿 입력'
os.environ['AWS_SESSION_TOKEN']    = '전달받은 토큰 입력'   # 임시 자격 증명인 경우만
os.environ['AWS_DEFAULT_REGION']    = 'us-east-1'          # 운영팀 지정 리전

import boto3
sts = boto3.client('sts')
print(sts.get_caller_identity())
⛔ 자주 발생하는 오류
  • InvalidClientTokenId → 키 복사 시 공백·줄바꿈 섞임. 다시 복사.
  • ExpiredToken → 임시 키가 이미 만료. 운영팀에 재발급 요청.

05

텍스트 모델 호출 (Claude)

Claude가 한국어 응답을 반환
import json
runtime = boto3.client('bedrock-runtime', region_name='us-east-1')

body = json.dumps({
    "anthropic_version": "bedrock-2023-05-31",
    "max_tokens": 100,
    "messages": [{"role": "user",
                  "content": "한국어로 '안녕'이라고만 답해줘"}]
})
resp = runtime.invoke_model(
    modelId='anthropic.claude-3-haiku-20240307-v1:0',
    body=body
)
print(json.loads(resp['body'].read())['content'][0]['text'])
⛔ AccessDeniedException이 뜨면

해당 리전에서 Claude 모델 액세스가 승인되지 않은 상태입니다. 운영팀에 "Bedrock 콘솔 → Model access에서 Claude Haiku 승인 확인 부탁드립니다"라고 회신해 주세요.

07

샘플 PDF 업로드 및 chunk 분할

"페이지: N, 청크: M" 형식으로 출력

Colab 왼쪽 사이드바 → 폴더 아이콘 → 업로드 버튼으로 아무 PDF(5~20페이지 권장)를 업로드한 뒤 아래 코드 실행:

from langchain_community.document_loaders import PyPDFLoader
from langchain.text_splitter import RecursiveCharacterTextSplitter

loader = PyPDFLoader("/content/sample.pdf")   # 업로드한 파일명으로 변경
docs = loader.load()
splitter = RecursiveCharacterTextSplitter(chunk_size=500, chunk_overlap=50)
chunks = splitter.split_documents(docs)
print(f"페이지: {len(docs)}, 청크: {len(chunks)}")
⛔ 주의 — 회사 기밀 문서 업로드 금지

Colab은 Google 외부 서비스입니다. 테스트용으로는 공개 자료(공공기관 보고서 PDF, 위키 문서 등)를 사용하세요. 청크가 0으로 출력되면 스캔 이미지 기반 PDF입니다 — 텍스트 PDF로 교체.

08

FAISS 검색

관련 청크가 top-3로 출력
from langchain_aws import BedrockEmbeddings
from langchain_community.vectorstores import FAISS

embeddings = BedrockEmbeddings(
    client=runtime,
    model_id='amazon.titan-embed-text-v1'
)
db = FAISS.from_documents(chunks, embeddings)
results = db.similarity_search("이 문서의 핵심 내용은?", k=3)
for i, r in enumerate(results):
    print(f"--- top {i+1} ---\n{r.page_content[:150]}\n")

05 — SECURITY

보안 유의사항

반드시 지켜주세요

  • 회사 기밀·내부 문서를 Colab에 업로드하지 마세요. Colab은 Google 외부 서비스입니다. 테스트에는 공개 자료만 사용.
  • AWS Access Key를 노트북 셀에 평문으로 남기고 공유하지 마세요. 노트북을 공유할 때는 자격 증명 셀의 출력을 지우거나 변수 셀을 삭제.
  • 전달받은 임시 자격 증명은 교육 종료 후 자동 폐기됩니다. 따로 어딘가에 저장해두지 마세요.
  • 모델 응답에 실제 사내 정보를 입력해 테스트하지 마세요. 테스트용으로는 "안녕"처럼 의미 없는 프롬프트만 사용.
